GLASS (Gravitational Lensing Amplification Solver Suite) is a lensing modulation finder for non-spherical and spherical lens systems.

**INSTALLATION** :

Step I (git cloning, so that files move to your local):

git clone https://github.com/AniruddhaCIndia/glass.git

Step II (Find installer, it's in the same folder):

cd glass

Step III (Install glance):

pip install -e . (Any changes ypu make to GLANCE are accounted without reinstalling it again.)

Step IV (Update installation):

cd glass

git pull origin main
